Welcome to My Django App!

This is the home page of my Django app. If you are seeing this page, it means that the app is working correctly.

If you want to see all products, add url ( /api/products ) page.

If you want to see details of a specific product, add url ( /api/products/product_id )

If you want to create a specific product, add url ( /api/products/create)

If you want to delete a specific product, add url ( /api/products/product_id/delete )

If you want to update a specific product, add url ( /api/products/product_id/update )
